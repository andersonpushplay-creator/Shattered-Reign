# 🎮 Shattered Reign

2D beat 'em up prototype developed in Unity, focused on combat feel, responsiveness, and player feedback.

---

## 🎥 Gameplay

https://youtu.be/F65JWOnsAPQ

---

## ⚔️ Features

- Combo system (3-hit combo)
- Parry system with counter window
- Enemy AI (chase, attack, slot positioning)
- Hit feedback (camera shake, hit stop)
- Input buffer for smoother combat

---

## 🧠 Focus

This project focuses on creating satisfying combat mechanics inspired by classic beat 'em ups, emphasizing:

- Game feel
- Responsiveness
- Player feedback

---

## 🛠️ Tech

- Unity (C#)
